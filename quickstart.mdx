# Quickstart guide

Get the I/O Festival 2026 website up and running on your local machine in just a few steps.

## Prerequisites

Before you begin, ensure you have:

- **Node.js 18.18+** installed (tested with Node.js 24.13.1)
- **npm**, **yarn**, **pnpm**, or **bun** package manager
- A code editor (VS Code recommended)
- Git for cloning the repository

<Note>
This project uses Next.js 16.1.6 which requires Node.js 18.18 or higher.
</Note>

## Quick setup

<Steps>
  <Step title="Clone the repository">
    First, clone the project repository to your local machine:
    
    ```bash
    git clone <repository-url>
    cd it-competition-web
    ```
  </Step>

  <Step title="Install dependencies">
    Install all required packages using your preferred package manager:
    
    <CodeGroup>
      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```

      ```bash pnpm
      pnpm install
      ```

      ```bash bun
      bun install
      ```
    </CodeGroup>
    
    This will install all dependencies including:
    - Next.js 16.1.6
    - React 19.2.3
    - Framer Motion 12.29.2
    - Three.js 0.182.0
    - Tailwind CSS 4
    - And more...
  </Step>

  <Step title="Start the development server">
    Launch the Next.js development server:
    
    <CodeGroup>
      ```bash npm
      npm run dev
      ```

      ```bash yarn
      yarn dev
      ```

      ```bash pnpm
      pnpm dev
      ```

      ```bash bun
      bun dev
      ```
    </CodeGroup>
    
    The development server will start and you'll see output like:
    
    ```
    ▲ Next.js 16.1.6
    - Local:        http://localhost:3000
    - Ready in 2.5s
    ```
  </Step>

  <Step title="Open in your browser">
    Navigate to [http://localhost:3000](http://localhost:3000) in your web browser.
    
    You should see the I/O Festival 2026 homepage with:
    - Animated preloader
    - Hero section with countdown
    - Competition tracks showcase
    - Full cyberpunk-themed interface
  </Step>
</Steps>

## What's included?

Once running, you'll have access to:

- **Homepage** (`/`) - Full landing page with all sections
- **Competition pages**:
  - `/kompetisi/ui-ux` - UI/UX Design track details
  - `/kompetisi/web-dev` - Web Development track details
  - `/kompetisi/business-case` - Business Case track details
- **Requirements page** (`/kelengkapan`) - Competition requirements and guidelines

## Development features

The development server includes:

- **Hot Module Replacement (HMR)** - See changes instantly without full page reload
- **Fast Refresh** - Preserves component state during edits
- **TypeScript checking** - Real-time type error detection
- **React Compiler** - Optimized builds with experimental React Compiler

<Note>
  The page will auto-update as you edit files. Try modifying `src/app/page.tsx` to see live changes!
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port 3000 is already in use">
    If port 3000 is occupied, Next.js will automatically try the next available port (3001, 3002, etc.).
    
    To use a specific port:
    ```bash
    npm run dev -- -p 3001
    ```
  </Accordion>

  <Accordion title="Module not found errors">
    If you encounter module resolution errors:
    
    1. Delete `node_modules` and lock files:
       ```bash
       rm -rf node_modules package-lock.json
       ```
    2. Clear npm cache:
       ```bash
       npm cache clean --force
       ```
    3. Reinstall dependencies:
       ```bash
       npm install
       ```
  </Accordion>

  <Accordion title="Three.js or 3D graphics not rendering">
    3D elements require WebGL support. Ensure:
    - Your browser supports WebGL 2
    - Hardware acceleration is enabled in browser settings
    - Graphics drivers are up to date
  </Accordion>

  <Accordion title="Smooth scrolling not working">
    The Lenis smooth scroll library may not work in older browsers. For best experience, use:
    - Chrome/Edge 90+
    - Firefox 88+
    - Safari 14+
  </Accordion>
</AccordionGroup>
