# Welcome to I/O Festival 2026

I/O Festival 2026 is a cutting-edge web platform for Indonesia's premier national technology competition. Built with the latest web technologies, it showcases a futuristic, cyberpunk-inspired design that brings together visionaries, developers, and creators to push the boundaries of what's possible.

## What is I/O Festival 2026?

I/O Festival 2026 is a national technology competition featuring three main tracks:

- **UI/UX Design** - Design measured user interfaces and experiences that serve the wider community
- **Web Development** - Build functional web services that solve real-world problems
- **Business Case** - Create concise technology-based business plans with measurable financial returns

The competition is open to university students, high school students, and the general public, with the top five teams from each track competing in the Grand Final.

## Technology stack

This project leverages cutting-edge web technologies to deliver an immersive, performant experience:

- **Next.js 16** - React framework with App Router and React Server Components
- **React 19** - Latest React with experimental React Compiler enabled
- **TypeScript 5** - Type-safe development
- **Tailwind CSS 4** - Utility-first styling with glassmorphism effects
- **Framer Motion 12** - Smooth animations and transitions
- **Three.js & React Three Fiber** - 3D graphics and interactive elements
- **Lenis** - Smooth scroll experience
- **MapLibre GL** - Interactive location maps

## Design philosophy

The platform embraces a **futuristic, cyberpunk aesthetic** with:

- **Glassmorphism** - Frosted glass cards with backdrop blur effects
- **Neon accents** - Vibrant orange (#ff8c42), blue (#55D5E7), and purple (#a64dff) highlights
- **Dark theme** - Pure black background (#0A0A0A) for maximum contrast
- **Custom typography** - Raela Grotesque for headings, Plus Jakarta Sans for body text
- **Animated effects** - Star dust particles, noise overlay, smooth parallax scrolling
