Section components are the building blocks of the I/O Festival 2026 landing page. Each section serves a specific purpose and uses Framer Motion for scroll-triggered animations.

## Hero & Landing Sections

<AccordionGroup>
  <Accordion title="Hero" icon="house">
    
    ### Purpose
    
    The Hero component is the main landing section featuring the I/O Festival branding, description, call-to-action, and a live countdown timer.
    
    ### Key Features
    
    - **Glassmorphic Design**: Frosted glass card with gradient borders
    - **Animated 3D Elements**: Static positioned decorative images scattered throughout
    - **Color Glows**: Animated background glows with pure CSS keyframes
    - **Countdown Integration**: Embedded Countdown component
    - **Scroll Indicator**: Animated chevron prompting user to explore
    
    ### Import
    
    ```tsx
    import { Hero } from '@/components/sections/Hero';
    ```
    
    ### Usage Example
    
    ```tsx
    export default function HomePage() {
      return (
        <>
          <Hero />
          {/* Other sections */}
        </>
      );
    }
    ```
    
    ### Animation System
    
    CSS keyframe animations for background glows:
    ```css
    @keyframes hero-glow-1 {
      0%, 100% { transform: translate(0px, 0px); opacity: 0.35; }
      50% { transform: translate(30px, -20px); opacity: 0.5; }
    }
    ```
    
    Framer Motion content animations:
    ```tsx
    // Title
    initial={{ opacity: 0, y: 30 }}
    animate={{ opacity: 1, y: 0 }}
    transition={{ duration: 0.7 }}
    
    // Description
    initial={{ opacity: 0, y: 20 }}
    animate={{ opacity: 1, y: 0 }}
    transition={{ duration: 0.6, delay: 0.15 }}
    ```
    
    ### Layout Structure
    
    - **Container**: Full viewport height with centered content
    - **Grid**: 12-column layout (7 cols text, 5 cols countdown on desktop)
    - **Elements**: 3D decorative images positioned absolutely throughout
    
    ### Gradient Text Effect
    
    ```tsx
    className="bg-clip-text text-transparent"
    style={{
      backgroundImage: 'linear-gradient(135deg, #a64dff 0%, #ffffff 25%, #ff8c42 50%, #55D5E7 75%, #ffffff 100%)'
    }}
    ```
    
  </Accordion>

  <Accordion title="Countdown" icon="timer">
    
    ### Purpose
    
    A dynamic countdown timer showing time remaining until registration opens/closes, with multiple phase states.
    
    ### Key Features
    
    - **Multi-phase Logic**: Handles before_open, wave1, wave2, and closed states
    - **Real-time Updates**: Updates every second
    - **Timezone Aware**: Uses Jakarta time (WIB/UTC+7)
    - **Customizable Accent**: Optional accent color prop
    - **Glassmorphic Design**: Premium frosted glass aesthetic
    
    ### Props
    
    ```tsx
    interface CountdownProps {
      accentColor?: string; // Optional hex color for accents (default: theme colors)
    }
    ```
    
    ### Import
    
    ```tsx
    import { Countdown } from '@/components/sections/Countdown';
    ```
    
    ### Usage Examples
    
    ```tsx
    // Default theme colors
    <Countdown />
    
    // Custom accent color
    <Countdown accentColor="#ff8c42" />
    ```
    
    ### Phase States
    
    | Phase | Target Date | Label | Description |
    |-------|-------------|-------|-------------|
    | `before_open` | Mar 15, 2026 | "Pendaftaran dibuka dalam" | Before registration opens |
    | `wave1` | Apr 5, 2026 | "Early Bird ditutup dalam" | Early bird pricing period |
    | `wave2` | Apr 30, 2026 | "Pendaftaran ditutup dalam" | Final registration period |
    | `closed` | N/A | "Pendaftaran telah ditutup" | After deadline |
    
    ### Implementation
    
    ```tsx
    const OPEN_REG_DATE = new Date("2026-03-15T00:00:00+07:00");
    const CLOSE_REG_DATE = new Date("2026-04-30T23:59:59+07:00");
    
    // Updates every second
    useEffect(() => {
      const timer = setInterval(() => {
        setNow(new Date());
      }, 1000);
      return () => clearInterval(timer);
    }, []);
    ```
    
    ### Time Display Format
    
    Displays four units with leading zeros:
    - Days (Hari)
    - Hours (Jam)
    - Minutes (Menit)
    - Seconds (Detik)

    ### Usage Example
    
    ```tsx

    ```
    
    ### Animation Implementation
    
    ```tsx
    const count = useMotionValue(0);
    const roundedDisplay = useTransform(count, (latest) => {
      return "Rp " + new Intl.NumberFormat('id-ID').format(Math.round(latest));
    });
    
    useEffect(() => {
      if (isInView) {
        const controls = animate(count, 46000000, { 
          duration: 1.2, 
          ease: "easeOut"
        });
        return controls.stop;
      }
    }, [isInView, count]);
    ```

## Common Animation Pattern

Most section components use consistent Framer Motion viewport settings:

```tsx
whileInView={{ opacity: 1, y: 0 }}
viewport={{ 
  once: true,           // Animate only once
  amount: 0,            // Trigger immediately when any part enters
  margin: "0px 0px 800px 0px" // Extended bottom margin for early trigger
}}
```

This creates a smooth, performant scroll experience across all sections.
